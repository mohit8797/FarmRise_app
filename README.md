# FarmRise

FarmRise is a full-stack agriculture marketplace platform designed to connect organic farmers with consumers through a simple, modern web experience. The project focuses on making fresh produce discovery easier while giving farmers a digital channel to showcase and sell fruits, vegetables, and crops.

## Project Overview

This project was built to demonstrate end-to-end web development skills across frontend UI, backend API development, authentication, and database integration. FarmRise includes product browsing pages, user authentication, a responsive landing experience, and a structured backend for managing user access and application data.

## Key Features

- User signup and login with secure password hashing
- JWT-based authentication flow
- Responsive frontend built with HTML, CSS, and JavaScript
- Separate product pages for fruits, vegetables, and crops
- Dashboard and marketplace-related UI screens
- Backend API built with Express.js
- Database integration for user and application data
- Static asset handling for product and website media

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Authentication: JWT, bcryptjs
- Database: MySQL
- Other Tools: dotenv, cors, nodemailer
